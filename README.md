# crowdshipai-web

**CrowdShip** — a live-streaming platform for watching (and funding) people build
software in real time. Builders vibe-code on camera; backers spend coins to shape
what gets built; recruiters watch the talent think. We own the coin rail and take
a cut of the middle.

> The atomic primitive is a video stream with someone building in front of it.

This repo is at the **founding stage** — no product code yet.

📄 **Start with [`CLAUDE.md`](./CLAUDE.md)** — the founding document. It defines what
CrowdShip is, the coin/ledger model, the builder-owned monetization principle, the
crypto self-settling-obligations pillar, content policy, and the open decisions.
Read it before building anything.
